Диаграмма по предметам

Помощь

1. Для загрузки сформированного журнала и его последующего отображения:

1) Перейдите во вкладку "Загрузка журнала".
2) Нажмите на кнопку "Выберите файл" (Обращаем ваше внимание, что для загрузки принимаются файлы только с расширениями .csv .txt .xlsx).
3) Загруженный файл отобразится на той же странице в виде таблицы.

2. Для создания и/или редактирования журнала:

1) Перейдите во вкладку "Создание или редактирование журнала"
2) Если вы загрузили журнал на предыдущем шаге, то отобразится таблица с имеющимися данными.
3) Для добавления записи в журнал введите данные в текстовые поля и нажмите на кнопку "Добавить", в конец таблицы встроится строка с введенными данными.
4) Для редактирования информации нажмите на кнопку "Редактировать" в соответствующей строке. В появившемся окне введите информацию, которую хотите изменить и нажмите на кнопку "Продолжить". Последовательно вводите изменения в следующие текстовые поля. Если изменять данные не нужно, также нажмите "Продолжить". Кнопка "Отмена" очистит ячейку таблицы.
5) Если вам нужно удалить данные об ученике, нажмите на кнопку "Удалить". Данная операция исключает строку из таблицы.
6) Для сохранения журнала нажмите на кнопку "Скачать таблицу". Журнал будет загружен в форматах .csv .txt и .html. Ваш браузер запросит разрешение на загрузку нескольких файлов, нажмите "Разрешить".

3. Для просмотра статистики по студентам:

1) Перейдите во вкладку "Статистика в табличном виде по студентам".
2) Отобразится таблица с фамилией студента, средним баллом, медианой, количеством оценок, процентом учеников для каждой оценки.

4. Для просмотра статистики по группам:

1) Перейдите во вкладку "Статистика в табличном виде по группам".
2) Отобразится таблица по каждому предмету с группой, средней оценкой, медианой, количеством и процентом учеников для каждой оценки

5. Для просмотра статистики в графическом виде:

1) Нажмите на вкладку "Статистика в графическом виде"
2) Отобразится статистика в виде круговой диаграммы.
